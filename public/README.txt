Put static assets here. Anything in /public is served from the site root.

Examples:
  public/papers/vio.pdf        ->  /papers/vio.pdf   (link it from src/data/research.ts)
  public/robots/striker.jpg    ->  /robots/striker.jpg
  public/team/aarav.jpg        ->  /team/aarav.jpg

The site currently uses generated SVG/CSS visuals so it needs no images to run.
Swap in real photos/PDFs by adding files here and pointing the data files / cards at them.
